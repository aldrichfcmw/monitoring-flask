# monitoring-flask

### Instalasi

- Clone repository

```
git clone git@github.com:aldrichfcmw/monitoring-flask.git
```

### Push Github

- Add File

```
git add .
```

- Memberi Comment

```
git commit -m "pesan"
```

- Mengirim ke Github

```
git push -u origin main
```

### Pull Github

- Mengambil file dari Github

```
git pull
```
